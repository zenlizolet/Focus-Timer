# StudyTimer - Pomodoro Focus App 

Een Pomodoro timer met focus tracking en todo list.

##  Features

- ️ **Pomodoro Timer** - 50 min focus + 5 min pauze cyclus
-  **Focus Tracking** - Houdt bij hoeveel je in totaal hebt gefocust
-  **Todo List** - Minimale, transparante todo lijst
-  **Custom Wallpapers** - Kies je eigen achtergrond of gebruik de mooie gradient
-  **Custom Alerts** - Gebruik je eigen notificatie geluidje
-  **Fullscreen Mode** - Distraction-free werken
-  **Modern Design** - Cyber aesthetic met Bahnschrift font

##  Snelstart

### Uitvoerbaar Bestand Gebruiken

1. Ga naar: `bin\Release\net10.0-windows\win-x64\publish\`
2. Zoek: `StudyTimer.exe`
3. Dubbelklik om te starten!

### Custom Notificatie Geluidje (Optioneel)

1. Plaats een  `notification.mp3` OF `notification.wav` in dezelfde folder als StudyTimer.exe
2. Ondersteunde formaten: MP3, WAV
3. De app speelt automatisch je custom geluidje!

### Custom Wallpaper (Optioneel)

1. Start de app
2. Klik op instellingen rechtsboven
3. Klik "Kies wallpaper"
4. Selecteer je afbeelding

##  Sneltoetsen

- **ESC** of **F11** - Toggle fullscreen/window mode
- **Enter** - Voeg todo item toe (in todo textbox)

## Gebruik

1. **Focus sessie starten**: Klik op START
2. **Pauze**: Klik op PAUSE om even te pauzeren
3. **Reset**: Klik op RESET om opnieuw te beginnen
4. **Todo's**: Type in het veld onderaan de todo lijst en druk Enter

##  Instellingen

Klik op het ⚙ tandwiel icoon voor:
- Werk timer aanpassen (standaard 50 min)
- Pauze timer aanpassen (standaard 5 min)
- Auto-start pauze aan/uit
- Wallpaper kiezen
- Standaard gradient achtergrond terugzetten

## Design

- **Font**: Bahnschrift (modern cyber aesthetic)
- **Colors**: Donkerblauw gradient met cyan accenten
- **Style**: Transparante UI elementen met ronde hoeken
- **Dialogs**: Custom moderne popups

## Distributie

Het uitvoerbare bestand is volledig zelfstandig en bevat:
- .NET runtime
- Alle dependencies
- Alles in één enkel bestand

Je kunt `StudyTimer.exe` gewoon kopiëren naar elke Windows PC en uitvoeren!
